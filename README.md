## Installation

1. **Clone the Repo**:
   ```bash
   git clone https://github.com/heroin1337/Steam-Checker.git

2. **Open the `SteamChecker.sln` file** in Visual Studio.

3. **Build and run the project**.

   * Once the solution is opened in Visual Studio, click on **Build > Build Solution**.
   * After the build is completed, press **F5** or click **Debug > Start Debugging** to run the application.

---

## **Download Pre-Built Version**

If you don't want to build the project yourself, you can download the pre-built version directly from GitHub:

1. Go to the **[Releases page](https://github.com/heroin1337/Steam-Checker/releases)**.

2. Download the latest release for your platform.

3. Extract the downloaded files and run the executable.
